Improving Documenation

Tip

To edit existing pages, use the Edit on GitLab link on the top right corner of the page.

To add a new page to documentation please follow the guideline provided by MKdocs to add new page and link them within the documentation.

Note

  • Install MkDocs.
  • Clone the repo locally and switch to develop branch.
  • Using Terminal, navigate to /docs folder.
  • Run mkdocs build
  • commit and push
